    weibo.com macos x client

    Project maintained by wuyb Hosted on GitHub Pages — Theme by mattgraham

    A little bit of history.

    This project is my hobby work during my PhD study. I was learning iOS (mainly Object-C) and decided to develop a new weibo.com client. After first release, I realized that I don't have the time and efforts to maintain it myself. So I open sourced it here.

    Design Principal

    The UI/UX design is mostly borrowed from echofon for its simplicity, which is my favorite style. The first release is in fact a replica. This is why I am not satisfied so I re-designed it.

    Make Contributions

    It is highly appreciated if you can send some pull requests to make it better. Currently, only the backend is working.
